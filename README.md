Fiz a tabela no Supabase criando a tabela de produtos.
Criei a pasta do Backend, colocando o server.js.
Criei a pasta do frontend colocando o index.html e o style.css.
Instalei o npm init -y e o npm install express no backend.
No server.js coloquei as rotas e para iniciar o navegador.
Testei o servidor no Thunder rodando node server.js.
Instalei npm install cors dotenv @supabase/supabase-js para conectar o supabase.
Criei o arquivo .env e adicionei minhas chaves da tabela do supabase.
